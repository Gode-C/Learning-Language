CSci 2014 - Homework 1

Homework 1 -- due Wednesday January 27

This homework will be graded out of 100 points. It will count 5% of the grade. Please submit it from the moodle page.

For this assignment you need to submit the clojure session, showing all the functions you typed and the results.  A simple way of capturing a session is by using the "script" command in unix.  You tell Unix to send a copy of everything you type and everything the system types to the file output.text by using the command

% script output.text

When you are done, to stop recording a copy of the session you hit Control-D

% ^D

Since Unix traps all the characters, if you make any typing error the file will contain all the characters you typed, including backspace and corrections. 

  1. [10 points] Write a function which returns the largest integer which is a multiple of 7 in a given range, extremes included, and nil if there is no multiple of 7. The range should be expressed as low end and high end, with both ends positive integer numbers. It should work like this:
    (maxmul7 1 20) = 14
    (maxmul7 20 35) = 35
    (maxmul7 1 6) = nil
    (maxmul7 7 10) = 7
  2. [10 points] Write a function to sum the cubes of the integers in a given range, extremes included. The range should be expressed as low end and high end, with both ends positive integer numbers. It should work like this:
    (sumcube 1 3) = 36
  3. [15 points] Write a function that takes a vector of at least three elements and returns a vector (or a list) consisting of all the elements except the second one. It should work like this:
    (drop2nd [1 2 3 4])= [1 3 4] or (drop2nd [1 2 3 4])= (1 3 4)
  4. [25 points] Write a program to check if a positive integer is perfect. A number is perfect if it is equal to the sum of its divisors, including 1 but not including the number itself. It should work like this:
    (perfect 6) = true
    (perfect 10) = false
  5. [20 points] Write a function to compute the average of a sequence of numbers and one to compute the standard deviation. For the standard deviation use the sample standard deviation. They should work like this:
    (ave [1 2 3 4]) = 5/2 or (ave [1 2 3 4] = 2.5
    (stddev [1 2 3 4])=1.29
  6. [20 points] Write a function that takes as input an element and a vector and returns the position of the first occurrence of the element in the vector or nil if the element is not in the vector. It should work like this:
    (posel [10 20 30 40] 30) = 2
